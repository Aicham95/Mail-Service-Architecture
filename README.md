# 📧 Mail Server Architecture

## 📖 À propos

Ce dépôt regroupe les travaux réalisés autour de la conception, du déploiement et de l'administration d'un serveur de messagerie.

Le projet est développé dans le cadre du **Master 2 Réseaux, Sécurité et Systèmes Distribués (RSSD)** et couvre les aspects théoriques et pratiques des infrastructures de messagerie modernes.

L'objectif est de comprendre le fonctionnement d'un service de messagerie, son intégration dans une infrastructure d'entreprise et sa mise en œuvre sous Linux.

---

# 🎯 Objectifs

- Comprendre l'architecture d'un serveur de messagerie.
- Étudier les protocoles SMTP, POP3 et IMAP.
- Identifier les différents composants d'un système de messagerie.
- Concevoir une architecture adaptée à une organisation(Assemblé Générale).
- Déployer et configurer un serveur de messagerie sous Linux.
- Tester les échanges entre clients et serveur.
